# Dodge the Creepers

A 2D arcade game built as a hands-on introduction to Godot. Survive as long as you can by dodging the creepers.

## Tech stack

- **Godot 4.6** — game engine
- **GDScript** — scripting language

Developed following the official Godot [Your First 2D Game](https://docs.godotengine.org/en/stable/getting_started/first_2d_game/) tutorial.
